# AI Prompt Token Counter 🚀

A lightweight, privacy-first, free online developer tool to instantly estimate token counts, word counts, character metrics, and API costs for major Large Language Models (LLMs) like ChatGPT (GPT-4o), Claude 3.5, and Gemini.

Built entirely using semantic HTML5, CSS3, and vanilla JavaScript. 

## ✨ Features

* **Real-time Counter:** Updates instantly as you type or paste your prompt.
* **Cost Estimation:** Calculates dynamic pricing models based on standard GPT-4o input rates.
* **100% Privacy & Security:** **Zero server calls.** All string processing happens locally within the user's browser context via client-side JavaScript.
* **SEO Optimized:** Built-in JSON-LD web application schema markup, semantic tags (`<section>`, `<h3 >`), and mobile-responsive layouts.
* **Zero Dependencies:** Pure vanilla code requiring no package managers (`npm`/`yarn`) or compilation steps.

## 🛠️ Tech Stack

* **Frontend:** Vanilla HTML5 / CSS3 (System UI font stacks, modern CSS Flexbox layout)
* **Logic:** Vanilla JavaScript (Client-side execution)
* **SEO Engine:** JSON-LD WebApplication structured schema metadata
